# Sexy Canvas — Skill de Aplicação e Auditoria

## O que é

O Sexy Canvas é um framework de **14 eixos emocionais** que traduz a neurobiologia do consumo em estratégia prática. A tese central: decisões de compra são tomadas pelo sistema límbico (emoção, instinto) e só depois justificadas pelo neocórtex (lógica).

Os 14 eixos dividem-se em dois polos:
- **7 Pecados Capitais** → ação rápida, impulso, choque
- **7 Sensações da Criança Interior** → vínculo, fidelização, conforto

> Regra de ouro: nunca use todos os 14 com a mesma intensidade. Selecione **2–4 eixos** e crie **intersecções** (ex: choque + acolhimento = água + óleo).

---

## Três modos de operação

### MODO 1 — APLICAÇÃO (criar do zero ou melhorar)
Usado quando o usuário quer criar ou eletrizar um material.

### MODO 2 — AUDITORIA (analisar material existente)
Usado quando o usuário fornece um texto, proposta, site ou campanha para análise.

### MODO 3 — IMPLEMENTAÇÃO (workshop / sessão estratégica)
Usado quando o contexto é uma sessão estruturada — com equipe, múltiplas personas e ciclo completo de ideação → priorização → plano de ação.

## MODO 1 — Aplicação

### Passo 1 — Sexy Empathy (persona com profundidade)

Antes de selecionar eixos, faça o diagnóstico da persona. Pergunte ao usuário (ou infira a partir do contexto):

- Qual é a **dor secreta** que a persona não admite publicamente?
- Onde aparece o **medo de exclusão ou perda de status** na rotina dela?
- Qual **vitória** ela quer provar para os outros (ou para si mesma)?
- Em qual estágio de consciência ela está?
  - **Self 1** (preso à narrativa social, precisa quebrar barreiras)
  - **Self 2** (saiu da matrix, pronto para novas regras)
  - **Self 3** (já opera com liberdade, quer alavancas)

### Passo 2 — Seleção dos eixos

Com base na persona, selecione **2–4 eixos** combinando ao menos 1 pecado + 1 sensação da criança.

Consulte a referência completa em `references/eixos.md`.

**Combinações comprovadas:**
| Objetivo | Combinação recomendada |
|---|---|
| Copy de urgência | Inveja + Segurança |
| Proposta premium | Vaidade + Pertencimento |
| Produto de automação | Preguiça + Recompensa |
| Campanha de causa | Ira + Amor |
| Lançamento/produto novo | Curiosidade + Luxúria |

### Passo 3 — Tradução para o material

Para cada eixo selecionado, aplique nas camadas:

1. **Oferta** — o que está sendo prometido e como é embalado
2. **Copy / Narrativa** — headlines, argumentos, CTAs
3. **UX / Produto** — fluxo, microinterações, estrutura
4. **Distribuição** — canal, timing, formato

### Passo 4 — Verificação anti-risco

Antes de entregar, cheque os 7 "pecados fatais" listados em `references/riscos.md`.

## MODO 2 — Auditoria

Quando o usuário fornecer um material existente, siga este protocolo:

### 2.1 — Mapeamento (o que já existe)
Para cada um dos 14 eixos, identifique:
- **Ativado** (presente e funcionando)
- **Fraco** (presente mas subaproveitado)
- **Ausente** (oportunidade não explorada)

### 2.2 — Diagnóstico
- Quais eixos estão em conflito ou se cancelando?
- Existe intersecção pecado + criança? Se não, o material é raso.
- Há "pecados fatais" (hiper-promessa, vaidade corporativa, etc.)?

### 2.3 — Recomendações priorizadas
Liste no máximo **3 intervenções** em ordem de impacto:
1. Eixo a ativar ou fortalecer
2. Elemento concreto a mudar (headline, garantia, CTA, estrutura)
3. Risco a eliminar

## MODO 3 — Implementação (workshop / sessão estratégica)

Para conduzir uma sessão completa de Sexy Canvas com equipe. Leia `references/implementacao.md` ao entrar neste modo.

### Resumo do processo (7 etapas)

| Etapa | Nome | O que fazer |
|---|---|---|
| 1 | Sexy Team | Montar equipe com especialistas e decisores |
| 2 | Persona e Experiência | Definir personas e canvas por contexto |
| 3 | Sexy Canvas Atual | Mapear realidade atual (forças + fraquezas) |
| 4 | Sexy Spy | Canvas dos concorrentes e referências |
| 5 | Objetivos | Definir metas do trabalho |
| 6 | Sexystorm | Ideação livre nos 14 eixos, sem julgamento |
| 7 | Sexy Plan | Priorizar com Esforço x Sexy → responsáveis + prazos |

### Priorização: Diagrama Esforço x Sexy

Após o Sexystorm, classifique cada ideia em um dos 4 quadrantes (leia `references/priorizacao.md`):

- 🔥 **Super Sexy** (− esforço, + sexy) → execute primeiro
- 🎁 **Interessantes** (+ esforço, + sexy) → planeje com cuidado
- ⚙️ **Tapa-Buraco** (− esforço, − sexy) → só se necessário
- 🚫 **Brochantes** (+ esforço, − sexy) → elimine do plano
