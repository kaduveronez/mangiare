# Diagrama Esforço x Sexy — Priorização de Ideias

Após o Sexystorm (geração de ideias nos 14 eixos), use esta matriz para selecionar o que executar.

## Os 4 quadrantes

```
                    + SEXY
                       |
   SUPER SEXY          |        INTERESSANTES
   (prioridade 1)      |        (prioridade 2)
                       |
- ESFORÇO -------------|------------- + ESFORÇO
                       |
   TAPA-BURACO         |        BROCHANTES
   (complementar)      |        (evitar)
                       |
                    - SEXY
```

### 🔥 SUPER SEXY (− esforço, + sexy)
- Ações que demandam **menos recursos** (tempo e dinheiro)
- **Fáceis de executar** e com **alto impacto** para o público
- **→ Execute primeiro. São as quick wins do Canvas.**

### 🎁 INTERESSANTES (+ esforço, + sexy)
- Geram impacto e grandes experiências e resultados
- Mas exigem **muita aplicação de recursos**
- São boas apostas — precisam realmente **valer a pena**
- **→ Execute depois das Super Sexy, com planejamento.**

### ⚙️ TAPA-BURACO (− esforço, − sexy)
- Experiências não tão impactantes, mas que demandam pouco esforço
- Talvez sirvam para **complementar as ações principais** ou manutenções
- **→ Use só se necessário. Não priorize.**

### 🚫 BROCHANTES (+ esforço, − sexy)
- Grande esforço aplicado para gerar experiências de pouco impacto
- **→ Elimine do plano. Custo-benefício não justifica.**

---

## Como usar no processo

1. Após o Sexystorm, liste todas as ideias geradas
2. Para cada ideia, avalie: qual é o nível de esforço? qual é o potencial de eletrização?
3. Posicione no diagrama
4. Priorize: Super Sexy primeiro → Interessantes → Tapa-Buraco → elimine Brochantes
5. Leve as ideias selecionadas para o **Sexy Plan** (responsáveis + prazos)
